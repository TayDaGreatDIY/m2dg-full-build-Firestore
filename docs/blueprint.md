# **App Name**: Married 2 Da' Game (M2DG)

## Core Features:

- Profile Display: Displays user profile with avatar, username, and key stats.
- Community Feed: Shows updates from other players with activity, posts, court information.
- Court Discovery: Allows users to find and discover basketball courts.
- Leaderboard: Displays the ranking of players based on XP and streaks.
- Direct Messaging: Enables direct messaging between users.
- Training Log: Allows user to keep track of training data to gain XP.
- User Authentication: Allows user authentication
- Firebase Initialization: Initializes firebase app, database, and storage

## Style Guidelines:

- Background: Off-black (#0A0A0A) for a basketball tunnel vibe.
- Card Background: Dark gray (#1A1A1A) for modules and cards.
- Text: Off-white (#FAFAFA) for general readability.
- Primary Accent: Gold (#FFD95A) for highlights and premium elements.
- Secondary Accent: Orange (#FF6B00) for CTAs and active elements.
- Headings: 'Space Grotesk' sans-serif, bold, and track tight
- Body: 'Inter' sans-serif for clear readability
- Heroicons or lucide-react for crisp, scalable vector icons
- Mobile-first, max-width 768px. Bottom navigation, top header bars, card layouts.
- Subtle transition animations for state changes (e.g., button clicks, content loading)
